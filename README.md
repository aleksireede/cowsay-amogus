# Cowsay

Cowsay implemented using Rust.

## Installation

### Arch linux or Manjaro linux

#### Install with paru/yay

```bash
paru cowsay-bin # or
yay cowsay-bin
```

#### Install with aur

```bash
git clone https://github.com/fffzlfk/cowsay-bin && cd cowsay-bin
makepkg -si
```
